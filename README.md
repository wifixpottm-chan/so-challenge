# so-challenge

A Python project for data collection, visualization, and milestone tracking using uv for dependency management.

## Modules

- `data_fetcher.py`: Handles data collection operations
- `plotter.py`: Provides visualization capabilities
- `milestones.py`: Defines project milestones

## Testing

Run tests with:
```bash
uv run pytest
```

## Installation

```bash
uv sync
```
